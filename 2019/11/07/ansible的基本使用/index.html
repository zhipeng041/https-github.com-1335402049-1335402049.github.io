ansible的基本使用
发表于2019-11-07|更新于2020-08-05|运维
|字数总计:603|阅读时长:2分钟|阅读量:

本章主要介绍ansible基本模块的使用

Command

在远程主机执行命令，默认模块，可忽略-m选项

Shell

和command相似

Script

运行脚本

Copy

从控制端复制文件到被控端

Fetch

从被控端取文件到控制端

File

设置文件属性

Hostname

管理主机名

Cron

计划任务，支持时间：minute,hour,day,month,weekday

Yum

管理包

ansible-doc

ansible模块文档说明

ansible-galaxy

连接https://galaxy.ansible.com下载相应的roles

ansible-console

可交互执行命令，支持tab
